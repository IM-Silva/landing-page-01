🎵 Raiz do Samba - Landing Page

Projeto de uma Landing Page responsiva desenvolvida para um clube de assinatura de camisas inspiradas nas escolas de samba do Brasil.

🚀 Objetivo

Apresentar o serviço, destacar benefícios, planos de assinatura e incentivar o cadastro do usuário através de formulário.

🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Grid Layout
- Media Queries
- Animações CSS

✨ Funcionalidades

- Layout responsivo para desktop e mobile
- Cards interativos na seção de planos
- Galeria adaptada para celular
- Formulário com validação
- Página de confirmação após envio
- Identidade visual personalizada

📱 Responsividade

Projeto adaptado para diferentes tamanhos de tela, com foco principal em desktop e dispositivos móveis.
